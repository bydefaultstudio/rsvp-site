Buttons are interactive elements that size to content by default.

Base Button

.button (or button element) provides the base button styling.

Styles

  • Font: Primary font, var(--button-font-size) (14px)
  • Font weight: var(--font-weight-bold) (700)
  • Text transform: Uppercase
  • Display: flex (inline-block fallback)
  • Padding: var(--space-l) vertical, var(--space-2xl) horizontal
  • Background: var(--button-primary)
  • Color: var(--button-text)
  • Border: var(--border-m) solid var(--button-primary)

Important

  • Buttons size to content and do not stretch full width by default.
  • Buttons use flexbox for internal alignment.

Button Modifiers

Class Effect
.is-outline Transparent background, colored border and text
.is-faded Faded background using var(--button-faded)
.is-small Smaller font size and padding
.is-icon Circular button with equal padding

Usage

<button class="button">Default Button</button>
<button class="button is-outline">Outline Button</button>
<button class="button is-faded">Faded Button</button>
<button class="button is-small">Small Button</button>
<button class="button is-icon">
  <div class="icn-svg"><!-- SVG icon --></div>
</button>

Modifier Combinations

  • .is-outline.is-faded - Outline button with faded border color

Background Utilities

Background utilities apply background color only. They do not affect layout, spacing, or sizing.

Class Background Color
.bg-primary var(--background-primary)
.bg-secondary var(--background-secondary)
.bg-plain var(--background-plain)
.bg-faded var(--background-faded)

Usage

<div class="bg-primary">
  <!-- Content with primary background -->
</div>

Important

  • These utilities apply background color only.
  • They do not affect layout, spacing, or sizing.
  • Combine with other layout and spacing utilities as needed.
